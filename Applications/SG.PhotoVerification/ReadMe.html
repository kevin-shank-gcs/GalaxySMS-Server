To get the new project up and running, the following steps must be performed..

  1. Generate a new GUID for the Assembly GUID
  2. Fix up the References. There FluentValidation and Portable.License packages must be refreshed. You will have to delete the entries from the packages.config file, then delete the references and get the current version from NuGet
  3. The required dll files are in the ReferencedAssemblies folder. Copy the dll files from the ReferencedAssemblies folder to the appropriate bin folder
  4. Update the Assembly attributes to reflect the appropriate values
  5. Rename the license file to match the exe name but with a 'license' extension
Happy coding!!!!
